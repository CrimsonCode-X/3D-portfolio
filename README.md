An immersive, production‑ready personal portfolio built with **Three.js**, **TailwindCSS**, and **GSAP** — all in a single HTML file.

## ✨ Features

- **3D Interactive Scene** – Floating shapes, particles, and a fully rigged low‑poly robot avatar with idle animations (breathing, blinking, head tilt, arm sway).
- **Glassmorphism UI** – Modern frosted‑glass cards with purple/violet accents on a dark background.
- **Responsive & Mobile‑First** – Adapts seamlessly to all screen sizes.
- **Smooth Navigation** – Active dot indicators and GSAP scroll animations.
- **Portfolio Sections**:
  - 🏠 Hero with 3D avatar
  - 👤 About & education timeline
  - 💼 6 projects with live GitHub links & tech tags
  - 🛠 Icon‑driven skills grid (languages, tools, libraries, databases, concepts)
  - 🏆 Achievements (hackathon rank, LeetCode stats, Forage virtual job simulations)
  - 📬 Contact with social links & email
- **Loading Screen** – Disappears after assets load.
- **Single HTML File** – No build step required. Deploy anywhere instantly.

## 📦 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/CrimsonCode-X/3D-portfolio.git
cd 3D-portfolio
```

### 2. Run locally
Simply open `index.html` in your browser:
```bash
# macOS
open index.html

# Windows
start index.html

# Linux
xdg-open index.html
```

Or use a local server (recommended for ES modules):
```bash
npx serve .
# visit http://localhost:3000
```

> **Note:** Three.js loads from CDN; an internet connection is required for the 3D scene.

## 🚀 Deployment

### Deploy to Vercel (Instant)
1. Drag & drop `index.html` into [vercel.com/new](https://vercel.com/new).
2. It will auto‑detect and deploy.

### Deploy via GitHub + Vercel
1. Push the repository to GitHub.
2. Import the repo in Vercel → automatic deployment on every push.

### Deploy to Netlify
1. Drag & drop the file into [app.netlify.com/drop](https://app.netlify.com/drop).
2. Or connect your GitHub repo.

**Important:** Ensure the file is named **exactly** `index.html` (lowercase) and placed in the repository root.
